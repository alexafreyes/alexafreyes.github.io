Open Draw Canvas Background 255 If mouse is clicked, draw droplet from top of window at same x position of mouse on screen. Mouse position acts as a force against the “droplet”, using the mouse, user can manipulate the movement of the droplet on the window. If mouse is clicked again, another droplet is drawn from its relative x position of the window. Droplet color is determined by position - pixels to values on screen. (x=hue, y=value)
